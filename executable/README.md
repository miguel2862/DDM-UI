# DDM Simulator Executable

This directory contains the executable installer for the DDM Simulator.

## Installation Instructions

1. Download `DDM_Setup.exe` (358 MB).
2. Run the installer and follow the on-screen instructions.
3. The installer will create a folder containing:
   - R portable
   - DDM.R file
   - Other necessary components

## System Requirements

- Windows 10 or later
- 4 GB RAM (8 GB recommended)
- 500 MB free disk space

## Usage

After installation:
1. Navigate to the installed directory.
2. Run the DDM Simulator shortcut or execute DDM.R using the provided R portable.

For detailed usage instructions, refer to the user manual in the main repository.

## Troubleshooting

If you encounter any issues, please check our FAQ in the main repository or open an issue on our GitHub page.
